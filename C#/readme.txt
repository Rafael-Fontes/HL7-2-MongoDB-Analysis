HL7toMongoDB is a simple Windows Application written in C# to store HL7 messages in MongoDB for analysis. 

This program is release under MIT license

This program uses official MongoDB driver for C# version 1.8.1




This program has been developed & tested on 
Windows 7 32 Bit
Visual Studio 2010
.NET Framework 4
MongoDB 2.4.3


Prerequisites
----------------
.NET Framework 4
MongoDB




Limitations
------------
This version doesn't perform any validation on HL7 messages


For HL7 message analysis follow articles

http://www.j4jayant.com/articles/hl7/33-hl7-analysis-mongodb-2
http://www.j4jayant.com/articles/hl7/32-hl7-analysis-mongodb
